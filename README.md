# 対角積数独パズル
学部3年の頃の研究室ゼミで作成

## 問題
3×3 のブロックに区切られた9×9の正方形の枠内に1∼9の数字を入れるパズル。
枠に入れる数字には以下の制約がある。
1. 空いているマスに1∼9のいずれかの数字を入れる。
2. 縦，横の各列に同じ数字が入ってはいけない。
3. 太枠で囲まれた3×3のブロック内に同じ数字が入ってはいけない。
4. 斜めに隣接する2つのマスの境界に表示されている数字は、その2つのマスの数字の積になる。
